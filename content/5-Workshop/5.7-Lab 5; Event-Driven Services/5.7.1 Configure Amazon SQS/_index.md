In this section, we will create a **Backup Vault** to store the backups generated by AWS Backup.

A Backup Vault is a centralized storage location for **Recovery Points** created from AWS resources. Using a Backup Vault makes it easier to manage backup data securely and supports data recovery whenever needed.

---

## Step 1. Open the AWS Backup Service

Sign in to the **AWS Management Console**.

Perform the following steps:

1. Search for **AWS Backup**.
2. Open the **AWS Backup** service.
3. Access the AWS Backup management console.

## Step 2. Create a Backup Vault

In the AWS Backup console:

1. Select **Backup vaults**.
2. Click **Create Backup vault**.
3. Enter the Backup Vault name.

Example:

```text
delivery-backup-vault
```

Then click **Create Backup vault** to complete the process.

## Step 3. Verify the Backup Vault

After the Backup Vault has been created successfully:

- Open the **Backup vaults** list.
- Verify that the new Backup Vault appears.
- Check the Backup Vault status.
- Confirm that the vault is ready to store recovery points.

The Backup Vault will be used in the following sections to store backup data for the system.
